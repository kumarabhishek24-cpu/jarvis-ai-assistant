# JARVIS AI Voice Assistant

JARVIS is a desktop voice assistant built using Python that can perform tasks through voice commands.

## Features
- Voice command recognition
- Opens applications
- Searches the web
- Provides voice responses
- Automates system tasks

## Technologies Used
- Python
- SpeechRecognition
- Pyttsx3
- Automation libraries

## Future Improvements
- Add ChatGPT integration
- Add smart home control
- Improve command accuracy
